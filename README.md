## 项目背景
> p2p理财平台项目,旨在提供给大家学习该平台中涉及到的技术框架、及核心业务。
    该项目分为多个子项目，其核心为金融公司：产品子项目和销售子项目；而第三方公司
    则开发自己的p2p理财平台项目；然后对接金融公司的销售项目进行理财产品的购买等
    操作。
